Paste Mock Test JSON

Full test object save/update karein, selected Test ID me questions append karein, mixed category questions array paste karein, ya 30 din ka category/date schedule ek saath import karein.

JSON Format Example

Single test:
{
  "title": "Daily Mock Test - Rajasthan GK",
  "category": "Rajasthan GK",
  "timeLimit": 20,
  "totalQuestions": 25,
  "status": "active",
  "date": "2026-05-24",
  "questions": [
    {
      "question": "राजस्थान की राजधानी क्या है?",
      "options": ["जयपुर", "जोधपुर", "उदयपुर", "कोटा"],
      "correctAnswer": "जयपुर",
      "explanation": "राजस्थान की राजधानी जयपुर है।"
    }
  ]
}

30 days/category-wise schedule:

  }
}
