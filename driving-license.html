Driving License Guide

ड्राइविंग लाइसेंस - पूरी जानकारी

Learner aur driving license ke liye identity, address aur age proof documents chahiye.

Form apply karne se pehle name, mobile number, Aadhaar detail aur documents ki spelling/checking zarur kar lein. Galat details se application delay ya reject ho sakti hai.

Tip: Original documents aur clear photo/scan copy dono ready rakhein.

Required Documents

  • Aadhaar Card
  • Address proof
  • Age proof
  • Photo
  • Mobile number
  • Blood group if required

Process

  1. Documents ready karein.
  2. E-MITRA WALA par contact karein ya nearest e-Mitra center visit karein.
  3. Application details verify karein.
  4. Application details aur reference number safe rakhein.
  5. Status update ke liye mobile number active rakhein.
